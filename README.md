# MyTimeTable

Transport timetable apps managed in one repository.

## Directory layout

- `apps/shinkansen/`: current Shinkansen web app
- `apps/bus/`: future bus web app
- `data/shinkansen/`: Shinkansen timetable data and archives
- `data/bus/`: future bus timetable data
- `scripts/shinkansen/`: Shinkansen data extraction scripts
- `scripts/bus/`: future bus data extraction scripts
